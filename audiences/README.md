# Pick your path in

Different people show up here for different reasons. Pick the door that fits —
each one is a complete onramp to participating.

> ⚠️ **Not medical advice.** Whatever door you walk through, read
> [docs/disclaimers.md](../docs/disclaimers.md) first.

## The common rules

Whichever door you walk through:

1. **No medical advice. Ever.** This repo is hypothesis-generation. Nothing
   here is a prescription.
2. **MIT-licensed, open, attribution-friendly.** Nothing you contribute will
   be patent-captured. Nothing we publish will be paywalled.
3. **Be honest about uncertainty.** Negative results are publishable here.
   Evidence weighting (`high` / `medium` / `low`) is in every dataset; use it.
4. **Pre-register before validating.** If you run a wet-lab campaign on
   compounds from this repo, file an experiment pre-registration (see
   [experiments/EXPERIMENT_TEMPLATE.md](../experiments/EXPERIMENT_TEMPLATE.md))
   before you unblind results. Stops cherry-picking.
5. **Safety first for patients.** Patient-reported triggers and case reports
   are anonymized by default. We never ask for PHI.
