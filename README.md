# MCAS.Opensource

**An open, hypothesis-generation engine for MCAS / MCAD rescue, maintenance, and remission compounds.**

> ⚠️ **Not medical advice.** This repository contains computational hypotheses and in silico predictions for research and educational purposes only. It is not a substitute for clinical care and must not be used to self-diagnose or self-treat. See [docs/disclaimers.md](docs/disclaimers.md).

---

## Mission

MCAS (Mast Cell Activation Syndrome) and MCAD (Mast Cell Activation Disorder) are
heterogeneous, under-served conditions with no cure and limited therapeutic options.
This project is a **public, MIT-licensed scaffold** for AI-assisted discovery of:

- **Rescue** compounds — acute mediator blockade and degranulation halt.
- **Maintenance** compounds — daily stabilization, mediator reduction, threshold raise.
- **Remission** compounds — upstream / root-cause modulators (Nrf2, KIT, MRGPRX2, oxidative & epigenetic injury).

Across pharma, herbs, supplements, and de novo AI-generated analogs. No paywalls,
no IP capture, no pharma gatekeeping. Every dataset, script, and prediction is in
this repo.

## Quickstart (this Mac)

```bash
# 1. Clone
git clone https://github.com/mrdulasolutions/MCAS.Opensource.git
cd MCAS.Opensource

# 2. Install local deps
pip install -e .

# 3. Build the master compound library (auto-fetches SMILES from PubChem)
python scripts/build_compound_library.py

# 4. Validate every SMILES with RDKit
python scripts/validate_smiles.py

# 5. Export for AI tooling
python scripts/export_for_ai.py

# 6. Open notebooks
jupyter lab notebooks/
```

The first two notebooks (`01_load_and_explore`, `02_qsar_deepchem`) run on CPU.
For the generative + docking notebooks (`03`, `04`) we recommend free Colab GPU —
each has an "Open in Colab" badge at the top.

## The data layer

| File | What it contains |
|------|------------------|
| [`data/compounds/seeds.json`](data/compounds/seeds.json) | Hand-curated anchor compounds (pharma + herbs + supplements) with PubChem CIDs, category, mechanism, evidence level |
| [`data/compounds/MCAS_Compound_Library_v1.csv`](data/compounds/MCAS_Compound_Library_v1.csv) | Generated by build script. SMILES + metadata for every compound |
| [`data/injury_mechanisms/MCAS_Injury_Mechanisms_v1.csv`](data/injury_mechanisms/MCAS_Injury_Mechanisms_v1.csv) | Oxidative, epigenetic, post-viral, gut, neurogenic, clonal, mediator-driven |
| [`data/triggers/MCAS_Triggers_v1.csv`](data/triggers/MCAS_Triggers_v1.csv) | Foods, smells / VOCs, physical, stress, infection, mold / mycotoxins |
| [`data/targets/MCAS_Targets.csv`](data/targets/MCAS_Targets.csv) | UniProt IDs for MRGPRX2, KIT, FcεRI, Syk, BTK, Nrf2, HRH1-4, CysLTR1 |
